# Tactac

Simple VSCode extension to move faster in code using shortcuts.


## Shortcut

|OS|Shortcut|
|---|---|
|macos| `shift+cmd+arrowup/down`|
|windows| `alt + arrow up/down`|


## Limitation

Based on regex, each languages are based on regex, current supported languages :

|lang|
|---|
|Markdown|
|Python|
|Json|
|Qml|
|Graphql|
